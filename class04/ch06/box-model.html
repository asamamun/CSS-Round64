CSS box model

CSS box model is the box that wraps the content of an element. Box model has 4 parts: content, padding, border, margin. The CSS box model consists of the following components: Content This is the content of an HTML element, such as a paragraph, image, <div>, or <span>. Padding Horizontal and vertical space surrounding the content. Border A border drawn around the padding. Margin Horizontal and vertical space outside the border.

Box-Sizing Property

The box-sizing property can change how the width is calculated:
content-box (default): The width property only sets the content width. Padding and border are added outside the width.
border-box: The width property includes the content, padding, and border. This makes it easier to manage the total width of an element.
