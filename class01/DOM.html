The DOM represents an HTML or XML document as a tree of nodes.

Each node is an object that represents a part of the document (e.g., an element, attribute, or text). 


Every part of the document is represented as a node in the DOM tree.

Types of nodes:

Element Node: Represents an HTML element (e.g., <div>, <p>).

Text Node: Represents the text content inside an element.

Attribute Node: Represents an attribute of an element (e.g., class, id).

Comment Node: Represents an HTML comment.

Document Node: Represents the entire document.
